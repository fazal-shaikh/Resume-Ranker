# 🚀 AI Resume Ranker

An intelligent web application that ranks resumes based on job descriptions using Natural Language Processing (NLP).

## 🔥 Features

* 📄 Upload multiple resumes (PDF)
* 🧠 NLP-based skill matching using spaCy
* 📊 Resume ranking based on similarity score
* ⬇ Download ranked results as CSV
* 🎨 Modern UI with interactive design

## 🛠 Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Python (Flask)
* **Libraries:** spaCy, Pandas, Scikit-learn
* **NLP Model:** en_core_web_sm

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/fazal-shaikh/Resume-Ranker.git
cd Resume-Ranker
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Download spaCy model

```bash
python -m spacy download en_core_web_sm
```

### 4. Run the application

```bash
python app.py
```

### 5. Open in browser

```
http://127.0.0.1:5000/
```

## 📸 Project Workflow

1. Enter job description
2. Upload resumes
3. System extracts text from PDFs
4. NLP processes and compares content
5. Ranks resumes based on relevance

## 📊 Output

* Ranked table of candidates
* Downloadable CSV file
